# Ejercicios

Ejercicios progresivos para practicar los contenidos de la guía.

---

## 1. Terminal básica

!!! example "Objetivo"

    Practicar rutas, creación de archivos, listado, redirección y búsqueda.

Tareas:

1. Crea una carpeta de laboratorio.
2. Entra en ella.
3. Crea tres archivos de texto.
4. Lista el contenido.
5. Guarda el listado en un archivo.
6. Busca una palabra dentro del listado.
7. Documenta qué comando hace cada cosa.

---

## 2. Linux administración

Tareas:

1. Mostrar versión del sistema.
2. Mostrar memoria disponible.
3. Mostrar uso de disco.
4. Mostrar procesos con más consumo.
5. Mostrar servicios fallidos.
6. Guardar resultados en un informe.

---

## 3. Windows PowerShell

Tareas:

1. Mostrar nombre del equipo.
2. Mostrar usuario actual.
3. Mostrar sistema operativo.
4. Mostrar procesos principales.
5. Mostrar servicios detenidos.
6. Exportar parte del resultado a un archivo.

---

## 4. Redes

Tareas:

1. Ver IP local.
2. Ver ruta por defecto.
3. Probar conectividad contra una IP externa.
4. Resolver un dominio.
5. Probar un puerto.
6. Escribir diagnóstico.

---

## 5. Usuarios y permisos

Tareas:

1. Identificar usuario actual.
2. Ver grupos del usuario.
3. Revisar permisos de una carpeta de prueba.
4. Explicar qué permisos existen.
5. Proponer una configuración segura.

---

## 6. Procesos, servicios y logs

Tareas:

1. Listar procesos.
2. Ordenar por consumo.
3. Revisar servicios.
4. Consultar logs o eventos.
5. Documentar una incidencia simulada.

---

## 7. Backups

Tareas:

1. Crear datos de prueba.
2. Crear copia.
3. Verificar copia.
4. Restaurar en otra carpeta.
5. Documentar el procedimiento.

---

## 8. Seguridad

Tareas:

1. Revisar usuarios.
2. Revisar grupos privilegiados.
3. Revisar servicios activos.
4. Revisar puertos abiertos.
5. Escribir recomendaciones defensivas.

---

## 9. Git

Tareas:

1. Crear repositorio local.
2. Crear README.
3. Hacer commit.
4. Crear rama.
5. Modificar archivo.
6. Ver diferencias.
7. Fusionar cambios.

---

## 10. Proyecto final inicial

Crea un informe que incluya:

- Sistema.
- Recursos.
- Red.
- Servicios.
- Logs.
- Usuarios.
- Backups.
- Conclusiones.
