# Plantillas de scripts

Plantillas didácticas para estructurar automatizaciones sencillas.

---

## Plantilla general

```text
nombre de la tarea
objetivo
entradas necesarias
pasos
salida esperada
verificación
errores posibles
mejoras futuras
```

---

## Recomendaciones

- Empieza con comandos manuales.
- Después agrúpalos en una plantilla.
- Añade parámetros solo cuando el flujo esté claro.
- Documenta objetivo, uso y salida.
- Guarda evidencias de prueba.
- Lleva la versión final a Script-Lab.
