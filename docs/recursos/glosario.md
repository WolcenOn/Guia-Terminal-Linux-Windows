# Glosario

Definiciones breves de conceptos frecuentes.

---

## Terminal

Interfaz que permite introducir comandos de texto.

## Shell

Programa que interpreta comandos. Ejemplos: Bash, PowerShell, CMD.

## Comando

Orden que se ejecuta en una terminal.

## Argumento

Dato sobre el que actúa un comando, como una ruta, archivo o servicio.

## Opción

Modificador que cambia el comportamiento de un comando.

## Ruta absoluta

Ruta completa desde la raíz del sistema o unidad.

## Ruta relativa

Ruta expresada desde la ubicación actual.

## Proceso

Programa en ejecución.

## Servicio

Proceso normalmente gestionado por el sistema para ofrecer una función continua.

## Log

Registro de eventos, errores o actividad.

## DNS

Sistema que traduce nombres de dominio a direcciones IP.

## Gateway

Puerta de enlace para salir a otras redes.

## Puerto

Número lógico usado para identificar servicios de red.

## Firewall

Sistema que filtra tráfico de red según reglas.

## Backup

Copia de seguridad de datos o configuraciones.

## Restauración

Proceso de recuperar datos desde una copia.

## Git

Sistema de control de versiones.

## Commit

Punto guardado en el historial de Git.

## Docker

Plataforma para ejecutar aplicaciones en contenedores.

## Contenedor

Entorno aislado que ejecuta una aplicación o servicio.

## Ansible

Herramienta de automatización e infraestructura como código.

## Runbook

Procedimiento documentado para resolver una tarea o incidencia.
