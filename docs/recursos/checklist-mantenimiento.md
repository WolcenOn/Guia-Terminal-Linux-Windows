# Checklist de mantenimiento

Lista de revisión periódica para sistemas Linux y Windows.

---

## Estado general

- [ ] Identificar equipo.
- [ ] Identificar usuario actual.
- [ ] Revisar fecha y hora.
- [ ] Revisar tiempo encendido.
- [ ] Confirmar sistema operativo.

---

## Recursos

- [ ] Revisar CPU.
- [ ] Revisar memoria.
- [ ] Revisar disco.
- [ ] Detectar procesos con mayor consumo.
- [ ] Revisar crecimiento anómalo de carpetas.

---

## Servicios

- [ ] Revisar servicios críticos.
- [ ] Detectar servicios fallidos.
- [ ] Revisar servicios configurados al inicio.
- [ ] Documentar servicios no esperados.

---

## Logs y eventos

- [ ] Revisar errores recientes.
- [ ] Revisar logs del servicio afectado.
- [ ] Guardar evidencias relevantes.
- [ ] Anotar fecha y contexto.

---

## Red

- [ ] Revisar IP local.
- [ ] Revisar gateway.
- [ ] Probar conectividad externa.
- [ ] Probar DNS.
- [ ] Revisar puertos relevantes.

---

## Usuarios y permisos

- [ ] Revisar usuarios locales.
- [ ] Revisar grupos privilegiados.
- [ ] Revisar permisos de carpetas críticas.
- [ ] Detectar cuentas antiguas o no utilizadas.

---

## Backups

- [ ] Confirmar existencia de copias.
- [ ] Confirmar fecha de última copia.
- [ ] Verificar tamaño coherente.
- [ ] Probar restauración en laboratorio.
- [ ] Documentar resultado.

---

## Seguridad

- [ ] Revisar actualizaciones pendientes.
- [ ] Revisar firewall.
- [ ] Revisar servicios expuestos.
- [ ] Revisar accesos administrativos.
- [ ] Documentar recomendaciones.

---

## Informe final

- [ ] Estado general resumido.
- [ ] Problemas encontrados.
- [ ] Evidencias incluidas.
- [ ] Recomendaciones.
- [ ] Próximas acciones.
