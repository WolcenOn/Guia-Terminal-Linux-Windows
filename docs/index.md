# Guía Terminal Linux & Windows

Bienvenida a la versión web pedagógica de la guía.

Esta documentación está pensada para aprender de forma progresiva: primero se entienden los conceptos, después se practican comandos, luego se crean scripts y finalmente se conectan los aprendizajes con proyectos profesionales.

## Cómo estudiar cada tema

!!! tip "Método recomendado"

    1. Lee el objetivo del tema.
    2. Consulta la búsqueda rápida si necesitas comandos.
    3. Abre el ejercicio correspondiente en Script-Lab.
    4. Ajusta bloques y genera código.
    5. Guarda evidencias.
    6. Documenta el procedimiento.
