Cada botón envía un identificador de ejercicio como parámetro `exercise`. Script-Lab valida ese identificador antes de cargar la práctica.

## Flujo recomendado

!!! tip "Trabajo en clase"

    1. Elige el ejercicio.
    2. Pulsa `Abrir ejercicio`.
    3. Script-Lab carga la práctica por ID seguro.
    4. Ajusta bloques y genera código.
    5. Escribe comandos, pseudocódigo o explicación.
    6. Revisa feedback.
    7. Copia la entrega sugerida.
