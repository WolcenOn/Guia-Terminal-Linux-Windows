# Glosario

Glosario básico de administración de sistemas, terminal, redes y scripting.

---

## Administración de sistemas

| Término | Definición |
|---|---|
| Sistema operativo | Software base que gestiona hardware, procesos, memoria, usuarios y archivos |
| Servicio | Programa que se ejecuta en segundo plano ofreciendo una función |
| Proceso | Programa en ejecución |
| PID | Identificador numérico de un proceso |
| Log | Registro de eventos del sistema o aplicación |
| Backup | Copia de seguridad |
| Restauración | Recuperación de datos desde un backup |
| Hardening | Endurecimiento de seguridad de un sistema |
| Monitorización | Supervisión del estado de sistemas y servicios |
| Runbook | Procedimiento documentado para una tarea o incidencia |

---

## Terminal y scripting

| Término | Definición |
|---|---|
| Terminal | Interfaz de texto para ejecutar comandos |
| Shell | Intérprete de comandos |
| Bash | Shell común en Linux |
| CMD | Intérprete clásico de Windows |
| PowerShell | Shell moderno orientado a objetos |
| Script | Archivo con comandos o instrucciones automatizadas |
| Variable | Nombre que almacena un valor |
| Parámetro | Dato que se pasa a un comando o script |
| Función | Bloque reutilizable de código |
| Código de salida | Número que indica si un comando terminó bien o con error |
| Redirección | Enviar salida o errores a un archivo u otro destino |
| Tubería | Pasar la salida de un comando a otro |

---

## Redes

| Término | Definición |
|---|---|
| IP | Dirección lógica de un equipo en red |
| Máscara | Define el tamaño de la red |
| Gateway | Puerta de salida hacia otras redes |
| DNS | Sistema que traduce nombres a IP |
| DHCP | Servicio que asigna configuración IP automáticamente |
| Puerto | Punto lógico donde escucha un servicio |
| Socket | Combinación de IP, puerto y protocolo |
| Firewall | Sistema que permite o bloquea tráfico |
| NAT | Traducción de direcciones de red |
| Latencia | Tiempo que tarda una comunicación |

---

## Seguridad

| Término | Definición |
|---|---|
| Mínimo privilegio | Dar solo los permisos necesarios |
| Autenticación | Verificación de identidad |
| Autorización | Permiso para realizar una acción |
| Cifrado | Protección de datos mediante algoritmos |
| Clave privada | Secreto usado para autenticación o cifrado |
| Secreto | Credencial, token o dato sensible |
| Auditoría | Revisión de acciones, accesos o configuraciones |
| Vulnerabilidad | Debilidad que puede ser explotada |
| Parche | Actualización que corrige fallos o vulnerabilidades |

---

## Git y Docker

| Término | Definición |
|---|---|
| Git | Sistema de control de versiones |
| Repositorio | Proyecto gestionado por Git |
| Commit | Cambio guardado en el historial |
| Rama | Línea independiente de trabajo |
| Pull request | Propuesta de cambios para revisar y fusionar |
| Docker | Plataforma de contenedores |
| Imagen | Plantilla para crear contenedores |
| Contenedor | Instancia ejecutándose de una imagen |
| Volumen | Almacenamiento persistente para contenedores |
| Compose | Herramienta para definir varios contenedores |
